## CURSO "INTRODUÇÃO AO BANCO DE DADOS"

### Propósito do Curso: 
Ao final do curso, o aluno será capaz de modelar um banco de dados relacional a partir de um conjunto de requisitos de dados. Também será capaz de utilizar a linguagem SQL para contruir e popular um banco de dados. Além disso, utilizará o SQL para responder questionamentos acerca dos dados que foram persistidos no banco. O curso irá utilizar o gerenciador de banco de dados _Microsoft SQL Server Express 2019_ como ferramenta de estudo. 
### SYLLABUS DO CURSO:
O curso será composto de .....aulas, cujos objetivos são os seguintes:
### AULA 1 – INTRODUÇÃO
Conhecer sumariamente o propósito de um BD relacional e sua utilidade na organização. Diferenciar um software de gerenciamento de banco de dados do próprio banco de dados. Baixar e instalar no seu computador o Microsoft SQL Server Express e a sua ferramenta de administração chamada de SQL Server Management Studio (SSMS). Conhecer os primeiros conceitos básicos sobre banco de dados: relações; tuplas; atributos e tipos de dados. Utilizar o SSMS para criar um banco e uma tabela. Inserir dados na tabela e fazer uma consulta simples. Se familiarizar com o uso da linguagem SQL.

### Aula 02  

Em 30 linhas definir o que é banco de dados, SGBD e sistema de banco de dados.
Responder para que serve um banco e descrever 3 principais caracteristicas.

- Descrição dos dados  
- Abstração e isolamento das operações e persistencia dos dados   
- Multiplas visões  

### Aula 03  

Introdução ao Projeto do banco de dados. Qual é o processo necessário para se elaborar um banco de dados. Do dado até a informação.
Requisitos -> Modelo conceitual (Diagrama de entidade relacionamento) -> Modelo Lógico
Descrição do modelo relacional. Modelo de Entidade-Relacionamento. Diagrama de Entidade-Relacionamento
Exemplo de modelagem  

### Aula 04

Tradução do modelo de entidade relacionamento em um projeto lógico de banco de dados.

### Aula 05

Algebra relacional, operações e SQL
